# Customer-Segmentation-Project

## Objective
Segment customers based on income and spending behavior using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm
K-Means Clustering

## Dataset
Mall Customers Dataset

## Features
- Data Analysis
- Data Visualization
- Elbow Method
- Customer Clustering
- Segment Visualization

## Output
Customer groups segmented into different clusters based on spending patterns.

## How to Run

Install libraries:

pip install -r requirements.txt

Run:

python customer_segmentation.py
